# VEX Aware Browser Extension Icons

This directory contains icon files for the VEX Aware browser extension in multiple sizes.

## Icon Sizes
- icon-16.png - 16x16 pixels (toolbar)
- icon-32.png - 32x32 pixels (extension management)  
- icon-48.png - 48x48 pixels (extension management)
- icon-128.png - 128x128 pixels (Chrome Web Store)

## Usage
These icons are referenced in the manifest.json file and used by the browser extension system.

For production, replace with actual VEX Aware branded icons.
