# Contact Us Submissions

This folder contains all contact form submissions from the VEX Aware website.

## File Structure

Each submission is saved as a JSON file with the following naming convention:
- `contact_YYYY-MM-DD_HH-MM-SS_randomId.json`

## File Contents

Each file contains:
- Submission timestamp
- Form data (name, email, company, subject, message, inquiry type)
- Unique submission ID
- IP address (if available)
- User agent information

## Privacy Note

These files contain personal information. Handle according to your privacy policy and data protection regulations.
